# CS3-Sentiment-Analysis-of-Restaurant-Reviews-Case-Study
## Overview
This project builds a machine learning pipeline to analyze and classify restaurant reviews from Yelp based on sentiment and star ratings. Using natural language processing (NLP) techniques, the goal is to understand how the language used in reviews relates to customer satisfaction.

The dataset contains over 20,000 restaurant reviews with varying star ratings and review lengths.
